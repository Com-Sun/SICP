- cond: if처럼생각

cond (술어) (결과식)

### if 와 cond의 차이

(if <조건식> <참> <거짓>)

- cond: 결과식 자리에 잇단식이 올 수도 있음. (술어)가 참일 떄 (결과식)이 잇단식이면 그 낱 식을 차례로 계산한 다음 마지막 식의 값을 cond값으로 돌려준다.

- if: (참) (거짓)자리에 식 하나만 가능
